Course Curriculum

  1. Introduction to Blockchain – What is blockchain, how it works, and key terminology
  2. Blockchain Structure – Blocks, chains, hash functions, and linking data securely
  3. Decentralization & Consensus – Peer-to-peer networks, mining, and proof-of-work
  4. Smart Contracts – Basics of Ethereum, smart contract functionality, and use-cases
  5. Blockchain Applications – Use in finance, supply chain, healthcare, and identity
  6. Final Project – Simulate a basic blockchain using Python or a visual demo
