Course Curriculum

  1. Introduction to Python – Syntax, variables, data types, and basic operations
  2. Control Flow – if-else statements, loops, and logical conditions
  3. Functions & Modules – Creating reusable code and using Python modules
  4. Data Structures – Lists, tuples, dictionaries, and sets
  5. Object-Oriented Programming – Classes, objects, inheritance, and encapsulation
  6. Final Project – Build a real-world Python application
